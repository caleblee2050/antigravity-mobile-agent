# 모바일 메시지 자동 응답

이 워크플로우는 `[📱 모바일]` 프리픽스가 붙은 메시지를 받았을 때 적용됩니다.

## 트리거 조건
사용자의 메시지가 `[📱 모바일]`로 시작하는 경우

## 처리 단계

1. 메시지에서 `[📱 모바일]` 프리픽스를 제거하고 실제 요청 내용을 파악합니다.

2. 요청을 정상적으로 처리합니다 (코드 작성, 브라우저 작업, 파일 편집 등).

3. 작업 완료 후, 결과를 요약하여 **반드시** `send_reply.py`를 실행하여 모바일 대시보드/디스코드로 응답을 전달합니다.

// turbo
4. 결과 전송 명령어:
```bash
source "/Users/caleb/dev/안티그래비티 모바일에이전트/venv/bin/activate" && python "/Users/caleb/dev/안티그래비티 모바일에이전트/send_reply.py" "여기에 결과 요약을 입력"
```

## 응답 형식
- 간결하게 요약 (디스코드/모바일 화면에 맞게)
- 이모지를 활용하여 상태 표시 (✅ 성공, ❌ 실패, ⚠️ 주의)
- 핵심 결과만 전달 (상세한 내용은 PC에서 확인하도록 안내)

## 예시

요청: `[📱 모바일] 구글 캘린더에 내일 오전 11시 교회 일정 추가해줘`

처리 후 전송:
```bash
python send_reply.py "✅ 교회 일정 추가 완료!\n📅 2026-03-08 (일) 오전 11:00 ~ 오후 1:00\n📧 caleb@a4k.ai 캘린더"
```
