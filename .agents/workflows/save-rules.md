# 자연어 기반 커스텀 룰 자동 저장 파이프라인

이 워크플로우는 사용자의 메시지에 "**이건 에이전트 룰이야**" 또는 "**이건 개발자 룰이야**"와 같은 트리거 문구가 포함되어 있을 때 가장 우선적으로 실행됩니다.

## 동작 기획 의도

일반 사용자도 복잡한 에이전트 설정이나 코딩 없이 일상 언어 한마디로 1초 만에 개인 비서를 커스텀할 수 있도록 고안되었습니다. 전역(비서 모드)과 지역(개발자 모드) 규칙을 스코프에 맞게 완벽히 분리하여, 컨텍스트 간섭 없이 깨끗한 AI 뇌 구조를 유지합니다.

## 처리 단계 체계

1. **지시문 파악 및 키워드 추출**
   - 사용자의 지시에서 핵심 규칙 내용을 도출합니다 (예: "앞으로 보고할 때는 이모지 쓰지마", "React는 함수형으로 짜").

2. **스코프 파악 및 규칙 영구 저장**
   - 🗣️ **"이건 에이전트 룰이야"** 조건 매칭 시:
     - **대상 파일**: `~/.gemini/antigravity/rules/agent_rules.md` (파일이 없으면 빈 폴더 및 파일 생성)
     - **로직**: 이 규칙은 모든 대화에 기본 장착되는 일상 비서(Global) 규칙입니다. 추출한 핵심 규칙을 위 마크다운 파일에 목록(`-`) 형태로 Append(추가) 합니다.

   - 🗣️ **"이건 개발자 룰이야"** 조건 매칭 시:
     - **대상 파일**: 현재 작업 중이거나 지시받은 타겟 프로젝트 폴더 최상단의 `.cursorrules` 또는 `dev_rules.md`
     - **로직**: 해당 폴더 내에서만 작동하는 엔지니어링 룰(코딩 컨벤션, Git 정책 등)로만 취급합니다. 해당 파일에 목록 형태로 Append(추가) 합니다.

3. **완료 피드백 전송 (가장 중요)**
   - 파일 저장이 끝나면 사용자가 안심할 수 있도록, 텔레그램으로 저장 완료 확인 알림을 전송해야 합니다.

// turbo
4. 결과 전송 명령어 예시:
```bash
source "/Users/caleb/dev/안티그래비티 모바일에이전트/venv/bin/activate" && python "/Users/caleb/dev/안티그래비티 모바일에이전트/send_reply.py" "✅ 에이전트 룰 장착 완료: 앞으로 보고 시 이모지를 사용하지 않겠습니다."
```
